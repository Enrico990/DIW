1) Antes de testar o site, abrir o "Abrir JsonServer.sh".

2) Em caso de erro verifique se voce possui o json server, se a resposta for nao e voce possuir o node.js use o comando "npm i -g json-server" na linha de comando para instalar ele, caso contrario voce vai ter que baixar o anterior separamente pelo link: https://nodejs.org, siga a instalacao e apos ela baixe o json-server como mostrado e tente abrir o "Abrir JsonServer.sh" novamente.

3) Se o seu json-server abriu normalmente e voce deseja fecha-lo use o comando "CTRL + C" como dito no terminal. Se voce apenas fechar o progama ele ira continuar aberto (no meu caso pelo menos), nessa situacao ele devera ser fechado pelo gerenciador de tarefas, voce deve procurar por "Node.js JavaScript Runtime" e encerrar o processo.

4) E isso :)
